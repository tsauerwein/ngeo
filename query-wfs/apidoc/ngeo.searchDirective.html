Class: searchDirective directive:ngeoSearch

ngeo.searchDirective

new searchDirective()

Provides the "ngeoSearch" directive, which uses Twitter's typeahead component to change an input text into a search field.

Example:

 <input type="text"
   ngeo-search="ctrl.typeaheadOptions"
   ngeo-search-datasets="ctrl.typeaheadDatasets"
   ngeo-search-listeners="crtl.typeaheadListeners">

See our live example: ../examples/search.html

HTML attributes:
Name Type Description
ngeo-search TypeaheadOptions The options.
ngeo-search-datasets Array.<TypeaheadDataset> The sources datasets.
ngeo-search-listeners ngeox.SearchDirectiveListeners The listeners.
